# AI-Powered Medical Image Diagnosis System

An intelligent healthcare platform that simplifies radiology image analysis using AI. The system processes CT scans, MRIs, and X-rays, interprets medical images accurately, and generates easy-to-understand diagnostic reports, bridging the gap between complex medical data and patient comprehension.

## Problem

* Patients receive reports full of technical terms they cannot interpret.
* Manual image analysis is time-consuming and prone to errors.
* Limited access to skilled radiologists in rural or low-resource areas.
* No unified system for AI-assisted medical insight generation.

## Solution

An AI-driven Medical Diagnosis System that processes medical images and delivers accurate, human-readable results.

* Accepts CT, MRI, or X-ray uploads directly.
* Uses Google multimodal AI for pattern detection and anomaly identification.
* Generates patient-friendly medical summaries instantly.
* Built with Streamlit for a smooth, interactive user experience.
